GET: /cart

GET: /cart/fulfillment/groups

View Fulfillment Groups for Order

Possible Response Codes:
400: Bad Request - The customer cannot be found on the request.
404: Not Found - Either the cart is null or this current cart does not have any fulfillment groups.

DELETE: /cart/fulfillment/groups

Query Parameters

  • Boolean: priceOrder (default true)

Remove All Fulfillment Groups from the Order

Possible Response Codes:
400: Bad Request - The customer cannot be found on the request.
404: Not Found - Either the cart is null or this current cart does not have any fulfillment groups.
500: Internal Server Error - A Pricing Exception has occurred while pricing the order.
